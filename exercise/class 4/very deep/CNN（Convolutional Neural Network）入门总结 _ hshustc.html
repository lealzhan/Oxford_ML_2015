CNN（Convolutional Neural Network）入门总结

发表于 2015 年 4 月 21 日 由 hshustc

    Deep Learning的三大框架:CNN（Convolutional Neural Network， 卷积神经网络）， DBN（Deep Belief Network, 深度置信网络）, AE(AutoEncoder， 自动编码机)。目前在CV领域应用最广的是CNN，本文结合作者自身的学习过程就CNN入门分享一些经验，供大家作为参考。实验室内部人员可在FTP上下载博文中提到的各种参考资料。

1. 学习CNN基本的概念
    比如卷积核、权值共享、pooling操作、激活函数等等。这在网上很多博客里面都可以找到，也可以参考作者写的本科论文（写的不好，不喜勿喷>_<）。
2. 了解人工神经网络的基本框架
    可以参考DeepLearningToolbox的matlab工具箱里面的CNN代码，对前向传导和误差反向传播（BP）做一定了解，对CNN网络的初始化建立、梯度检查、训练、测试等各个阶段有一定认识。
 
3. 学习最近流行的CNN的框架
    这里推荐几篇经典论文，这几篇论文也分别对应一个CNN的结构，但从原理上讲这些结构都是类似的：
    Convnet：《ImageNet Classiﬁcation with Deep CNN》
    改进的Convnet：《OverFeat： Integrated Recognition, Localization and Detection using Convolutional Networks》
    Caffe: 《Caffe：convolutional architecture for fast feature embedding》
    NIN: 《Network In network》
    VGGNet：《Very deep convolutional networks for large-scale image recognition》
    GoogleNet: 《Going Deeper with convolutions》
    另外推荐FeiFei Li等的公开课“Convolutional Neural Network for Visual Recognition”的讲义，现在网上还找不到相应的视频。这些讲义讲的非常好，里面介绍的基本都是最新的成果。
 
4. 读代码， 跑代码（推荐caffe）
    目前实验室服务器上配置有有convnet和caffe，可以自己动手运行一下（请拷贝一份到自己的工作目录下）。
    Convnet Documentation：https://code.google.com/p/cuda-convnet/
    Caffe Tutorial: http://caffe.berkeleyvision.org/tutorial/
    这两个网站上有Convnet和Caffe的说明文档以及运行方法，请务必仔细阅读，不能仅满足跑示例！
    这两个代码是最常见的，Convnet有一个升级版Convnet2，严格来讲convnet的结构和论文《ImageNet Classiﬁcation with Deep CNN》并不是完全对应，比如说Convnet中不含有dropout层，而Convnet2中则含有（当然也可以自己加，代码都是很灵活的）。另外Convnet2支持多GPU训练。此外还有Theano以及Torch7等，这两个作者接触的不多，有兴趣的可以看一看。
    另外不要害怕去读代码。虽然代码量比较大，但适当地阅读会加深你对代码结构和CNN的理解。由曹师兄主要负责完成的Convnet和Caffe的代码分析也可以在FTP上找到，在此对曹师兄表示衷心的感谢！
 
5. 了解如何应用CNN
    目前CNN几乎在CV的各个领域都取得了state-of-art的效果，如classification, localization, segmentation， detection等等，可以结合自己的兴趣寻找相关的文章来阅读。在这里就不一一赘述了。
 
6. 自己动手（丰衣足食）
   入门小白一枚>_<, 目前也正在摸索，共同努力吧！
